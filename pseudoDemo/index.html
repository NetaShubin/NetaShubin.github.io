Pseudo Classes and Elements:

Let's dive deeper into CSS and how it works

If we think of CSS in terms of selectors (the tags), properties (font size, padding, etc.), and values (what you set the properties to), then a typical CSS entry would look like this

            
                selector {property: value;}

To style the above, we need to add 2 tags together. We call this CSS Combinators

Pseudo Classes:

If we think of CSS in terms of selectors, properties, and values, then pseudo class is an add-on to a selector

            
                selector:class { property: value;}

Can be used for links

If we want out link to look different, we can add...

            a {color: green;}

We can also make our link 2 look different in diff scenarios...

  • :hover
  • :active
  • :visited

But we can also make a new link look different by giving it a class

We can use Pseudo Classes on any tag

Here we will add a hover status to our div

Anything that has a selector you can add a hover to

We can even call on specific selectors depending on their heirarchy (ex: first-child or last-child)

You can also create tooltips with pseudo classes

hover over me

This is your hidden tooltip

Pseudo Elements:

If we think of pseudo classes as add-ons, then pseudo elements are creating new elements from a selector, Ex: change the last letter, etc.

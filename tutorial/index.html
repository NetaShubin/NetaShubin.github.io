Let's recall some javascript!

How do you create a javascript function and link it to an element in HTML?

  1. Give your element an id
  2. Create a button or some sort of action to trigger the javascript function
  3. Add the following attribute and value to the element: 
                    
                    onclick = "function_name()"
  4. In your javascript file (don't forget to link it to your html page), create some function: 
                    
                    function function_name() {
                        // Specific action here
                        // Don't forget semi-colons
                    }
  5. Get the element you want to change: 
                    
                    let el = document.getElementById("your_id");
  6. You can change its text (if it is a p tag for example) by saying: 
                    
                    el.innerHTML = "added text here";

Date and Time

  1. Create a p tag and give it an id called "date" 
                    
                    <p id = "date"><p>
  2. Create a button that says "Get Date" and give it an onclick attribute 
                    
                    <button onclick="getDate()">Get Date<button>
    This means that when the button is clicked, it will call a getDate() function which we will create next
  3. Now let's go to our script.js file and write the following: 
                    
                    function getDate() {
                        // Get the "date" element we made in step 1
                        let text = document.getElementById("date");
                        // Create and set variable to current date and time using the Date() function
                        const now = new Date();
                        // Set the content inside the p tag to the date 
                        text.innerHTML = now;
                    }

What else can we do?

  • You can get the timestamp in javascript with 
                    
                    now.getTime();
    The timestamp is a value consisting of the number of milliseconds that have passed since midnight on January 1st, 1970! This day is called Epoch.
  • You can get the date given a certain timestamp 
                    
                    new Date(timestamp);
  • You can also create the date variable based on a specified date and time 
                    
                    new Date(year, month, day, hours, minutes, seconds, milliseconds);
  • You can create your own date variable to, for example, store a birthday 
                    
                    // Initialize a new birthday instance
                    const birthday = new Date(1980, 6, 31);
  • You can also retrieve certain parts of the date with different get() functions 
                    
                    // Get year
                    getFullYear();

                    // Get month (0-11, where 0 is January)
                    getMonth();

                    // Get the day of the month (1-31)
                    getDate();

                    // Day of the week (0-6, where 0 is Sunday)
                    getDay();

                    // Get the hour (0-23, where 0 is midnight)
                    getHours();

                    // Get the minutes (0-59)
                    getMinutes();

                    // Get seconds (0-59)
                    getSeconds();

                    // Get milliseconds (0-999)
                    getMilliseconds();

                    // Get the timestamp (milliseconds since Epoch time)
                    getTime();
    The syntax for the above for, let's say, a date variable called myDate would be 
                    
                    myDate.getFullMonth();

Try writing the code that checks if today is October 3rd

Is today October 3rd?

Solution:

        
            <p id="practice">Is today October 3rd?<p>
            <button onclick = "check()">Let's Check!<button>
        
        
            function check() {
                let answer = document.getElementById("practice");
            
                // Get today's date
                const today = new Date();
            
                // Compare today with October 3rd
                if (today.getDate() === 3 && today.getMonth() === 9) {
                    answer.innerHTML += "It's October 3rd";
                } else {
                    answer.innerHTML += "It's not October 3rd";
                }
            }
